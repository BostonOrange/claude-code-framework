## How to Use This Index

1. **Before any doc fetch:** Search this index for the library/topic
2. **Cache hit:** Read the referenced `.md` file directly — skip the fetch
3. **Cache miss:** Fetch docs, save to a new file, add a row to this index
4. **Partial hit:** Topic exists but doesn't cover what you need — fetch the missing part, merge into the existing file, update the topics column here
5. **Stale entry:** If `Last Updated` is older than 30 days and topic is actively used — re-fetch, merge, update the date here
